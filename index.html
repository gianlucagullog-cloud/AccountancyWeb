🛠 Come ottenere il Client ID (5 minuti)

1Vai su console.cloud.google.com e crea un nuovo progetto (es. InvoiceRegister)
2Vai su API & Services → Library e abilita Google Drive API
3Vai su API & Services → OAuth consent screen → scegli External → compila nome app → aggiungi il tuo email come test user
4Vai su Credentials → Create Credentials → OAuth client ID → tipo Web application
5In Authorized JavaScript origins aggiungi esattamente: http://localhost:8080
6Copia il Client ID generato e incollalo nel campo qui sopra
7Apri il file HTML con: python3 -m http.server 8080 poi vai su http://localhost:8080/registro_fatture_13.html
8Per il Folder ID: apri la cartella Drive nel browser → copia l'ID dall'URL dopo /folders/
⚠️ Il file deve essere aperto via http://localhost:8080 (non da file://) altrimenti il popup OAuth viene bloccato dal browser.
